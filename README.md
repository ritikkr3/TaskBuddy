# ✨ TaskFlow - Advanced Todo App

## 📌 Overview

TaskFlow is a modern and responsive Todo App built using React.js and Tailwind CSS. It helps users manage daily tasks efficiently with powerful features like search, filtering, editing, and dark mode.

---

## 🚀 Features

* ➕ Add Tasks
* ✏️ Edit Tasks
* 🗑️ Delete Tasks
* ✅ Mark as Completed
* 🔍 Search Tasks
* 🎯 Filter (All / Completed / Pending)
* 🌙 Dark Mode
* 💾 Local Storage Support
* 📱 Fully Responsive UI

---

## 🛠️ Tech Stack

* Frontend: React.js
* Styling: Tailwind CSS
* State Management: React Hooks
* Storage: LocalStorage

## ⚙️ Installation & Setup

1. Clone the repository:

```id="j1gb9e"
git clone https://github.com/your-username/taskflow.git
```

2. Navigate to project folder:

```id="j5scb0"
cd taskflow
```

3. Install dependencies:

```id="chfrsq"
npm install
```

4. Start the app:

```id="htncib"
npm run dev
```
